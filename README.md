# unoconv-node

A base environment for GradApp/doc2pdf

## Build

```
$ docker build -t unoconv-node .
```

## Usage

Intended primarily as a base image from which to create other application images. Eg.

```Dockerfile
FROM unoconv-node

# Do something

```
